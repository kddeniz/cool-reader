Cool Reader: service description

Cool Reader is a static, client-only web application: markdown editing, live HTML preview, and file open/download happen in the browser. This site does not host a public REST or GraphQL API.

Machine-readable metadata

Structured data: schema-ld.json (JSON-LD) — suitable for the rel="describedby" link from the home page.
